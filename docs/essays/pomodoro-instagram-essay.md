# AI와 일하다 번아웃이 왔다

---

새벽 3시. 커피는 세 번째.

"딱 이것만 고치고 자야지"가 세 시간째다.

Claude와 코딩하기 시작하면서 생긴 버릇이다. 질문하면 바로 답이 온다. 실행하면 바로 결과가 나온다. 막히는 순간이 없다. 그래서 멈출 수가 없다.

게임 중독이랑 비슷하다는 걸 깨달았다. 끊임없이 퀘스트가 주어지고, 레벨업 알림이 뜨고, "한 판만 더"가 반복되는 그 느낌. 다만 이건 일이니까 죄책감도 없다. "열심히 하는 거잖아"라고 합리화하면서.

몸이 먼저 신호를 보냈다. 어깨 통증, 두통, 집중력 저하. 분명 더 많이 일하는데 결과물은 그만큼 안 나왔다.

---

해결책은 의외로 단순했다.

**나를 강제로 멈추게 하는 시스템.**

뽀모도로 앱은 있었다. 하지만 내가 무시하면 끝이었다. 그래서 Claude Code의 훅 시스템으로 직접 만들었다. 25분 작업하면 화면이 잠긴다. 물리적으로. 5분 휴식 후에야 다시 시작할 수 있다.

"한 번만 더"를 위한 유예 기능도 넣었다. 대신 연속 2번까지만. 그 이상은 시스템이 거부한다.

---

처음엔 답답했다. "아 진짜 지금 이거 끝내야 하는데!"

그런데 신기하게도, 강제로 쉬고 나면 더 잘 풀렸다. 막혀있던 문제가 휴식 후에 "아, 이거였네" 하고 해결되는 경우가 많았다.

뇌는 쉬어야 정리가 된다는 걸, 머리로는 알았지만 몸으로 체감한 건 이때가 처음이었다.

---

지금은 하루 4-6 뽀모도로를 목표로 한다. 2-3시간 집중. 그 이상은 시스템이 경고를 띄운다. "충분히 했어, 내일 해."

아이러니하다. AI가 나를 더 빠르게 달리게 해서 생긴 문제를, AI가 나를 멈추게 해서 해결했다.

도구는 중립적이다. 어떻게 쓰느냐가 문제다.

---

*P.S. 이 글도 뽀모도로 2개 안에 끝냈다. 예전 같으면 새벽까지 다듬고 있었을 텐데.*

## 🎨 스크린샷 촬영 가이드

터미널에서 다음 명령어를 순서대로 실행하여 스크린샷 촬영:

```bash
# 1. 작업 중 상태 (임시로 상태 파일 수정 필요)
pomo-status

# 2. 휴식 시작 화면
pomo-break

# 3. 목표 달성 상태
pomo-status

# 4. 세션 종료 요약
pomo-stop
```

**촬영 팁:**
- 터미널 배경을 어둡게 (다크 모드)
- 폰트 크기 크게 (가독성)
- 불필요한 정보는 크롭
- 이모지가 잘 보이는지 확인
