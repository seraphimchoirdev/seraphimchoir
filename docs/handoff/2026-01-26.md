## 9. Potential Gotchas

> 스크린샷 캡처 과정에서 발생한 이슈입니다.

- ⚠️ **Playwright MCP vs CLI**: MCP 브라우저 도구는 기존 Chrome 세션과 충돌함 → `npx playwright screenshot` CLI 사용 권장
- ⚠️ **Playwright 브라우저 설치**: `npx playwright install chromium` 먼저 실행 필요
- ⚠️ **한글 렌더링**: 실제 터미널 캡처 시 폰트 문제 가능 → HTML 템플릿이 안전
